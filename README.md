# 100 Days ML Algorithm Challenge

A structured 100-day learning challenge to master machine learning and AI algorithms from fundamentals to cutting-edge models.

## Overview
This repository follows a day-by-day curriculum across six phases:
- Classical Machine Learning foundations
- Neural network fundamentals
- Convolutional neural networks
- Advanced computer vision
- Sequence models and NLP
- Cutting-edge AI and final integration

The goal is to build strong algorithmic understanding and practical implementation skills by implementing or studying one concept each day.
